Section 03

Keyboard Events

Découvre comment JavaScript écoute lorsque l’utilisateur presse et relâche les touches du clavier.

Explanation

Les Keyboard events se produisent lorsque l’utilisateur appuie sur des touches et JavaScript réagit instantanément.

Dans cette section, tu vas tester keydown et keyup.

keydown se déclenche quand une touche est pressée.
keyup se déclenche quand la touche est relâchée.

JavaScript peut aussi détecter des special keys comme Ctrl, Shift, Enter et Escape.

Ces keys sont utiles pour créer des shortcuts et des smart actions.
